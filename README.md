# DadClock - Screen Time Management

DadClock is a screen time tracking and scheduling application that helps you or someone you care for maintain a healthy balance between digital usage and real life. Whether you're a parent wanting to manage your kid's app usage, or just trying to keep yourself accountable, DadClock makes it simple.

## What It Does

**Track Your Screen Time:** Get detailed insights into which apps and websites you're spending the most time on. See daily breakdowns, weekly trends, and top apps at a glance.

**Create Smart Schedules:** Set up custom time schedules to block specific apps or websites during certain hours. Need to focus on homework at 2 PM? Block social media. Want to enforce bedtime? Schedule a full app lockdown.

**Stay Accountable:** With real-time analytics and visual charts, you'll see exactly where your time goes. Sometimes that awareness alone is enough to change habits.

## Getting Started

First, make sure you have Node.js installed, then run:

```bash
npm install
npm run dev
```

Head to [http://localhost:3000](http://localhost:3000) to see the app in action.

## Tech Stack

Built with modern web tech:
- **Next.js 16** - React framework for fast, scalable web apps
- **Supabase** - Database and authentication
- **TypeScript** - For safer, more reliable code
- **Tailwind CSS** - Beautiful, responsive styling
